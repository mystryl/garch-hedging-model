## 一、模型修正总结

### 1.1 核心理论修正

| 修正项 | 修正前 | 修正后 | 影响 |
|--------|--------|--------|------|
| **协整窗口时序** | `t`时刻包含在估计窗口 | 使用`t-1`及之前数据 | ✅ 避免未来信息泄露 |
| **ECM方程量纲** | 对数收益率(%)与ect(元) | 价格差分(元)与ect(元) | ✅ 量纲统一,经济意义明确 |
| **套保比例公式** | `h_ecm × (σ_ecm/σ_f)²` | `Cov(ΔS,ΔF) / Var(ΔF)` | ✅ 理论严谨,最小方差推导 |
| **税点调整** | 直接除以1.13 | 改为可选,默认不调整 | ✅ 符合实务逻辑 |
| **异常值处理** | 硬编码[0,2] | 3σ原则+非负约束 | ✅ 统计合理性 |

### 1.2 新增技术实现

- ✅ **DCC-GARCH集成**: 基于mgarch库实现时变协方差估计
- ✅ **条件协方差提取**: `dcc_garch_model.py` 独立模块
- ✅ **滚动窗口优化**: 避免未来信息泄露的时序对齐

## 二、套保效果对比

### 2.1 综合性能指标

| 模型 | 方差降低比例 | 夏普比率 | 最大回撤 | Ederington指标 |
|------|-------------|---------|---------|---------------|
| **Basic GARCH** | **56.08%** | **0.0099** | **-33.18%** | **0.5608** |
| ECM-GARCH | 0.01% | 0.0003 | -53.01% | 0.0001 |
| DCC-GARCH | 51.92% | 0.0067 | -31.17% | 0.5192 |

### 2.2 关键发现

**✅ 基础GARCH模型表现最优**
- 方差降低: 56.08% (接近理论最优值)
- 夏普比率: 0.0099 (风险调整后收益最佳)
- 最大回撤: -33.18% (风险控制良好)

**⚠️ ECM-GARCH模型效果显著下降**
- 方差降低: 仅0.01% (几乎无套保效果)
- **原因分析**:
  1. 从对数收益率改为价格差分后,数值特征发生根本变化
  2. DCC-GARCH在价格差分框架下拟合质量下降
  3. 可能需要重新调整模型参数或估计方法

**✅ DCC-GARCH模型保持良好表现**
- 方差降低: 51.92% (略低于Basic GARCH但优于ECM-GARCH)
- 最大回撤: -31.17% (所有模型中最佳)
- 动态相关性捕捉有效

## 附录: 技术细节

### A. 修正前后对比

**协整窗口时序**:
```python
# 修正前 (错误)
for t in range(coint_window, T):
    spot_window = spot[t-coint_window:t]  # 包含t
    ect[t] = spot[t] - (β₀(t) + β₁(t) * futures[t])

# 修正后 (正确)
for t in range(coint_window+1, T):
    spot_window = spot[t-coint_window-1:t-1]  # 不包含t
    ect[t] = spot[t] - (β₀(t) + β₁(t) * futures[t])
```

**套保比例公式**:
```python
# 修正前 (无理论支撑)
h(t) = h_ecm × (σ_ecm(t) / σ_f(t))²

# 修正后 (最小方差推导)
h*(t) = Cov(ΔS_t, ΔF_t) / Var(ΔF_t)
```
