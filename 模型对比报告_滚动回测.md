## 核心差异分析

### 1. 方差降低效果（套保有效性）

**排序**: DCC-GARCH (85.03%) > Basic GARCH (84.24%) ≫ ECM-DCC-GARCH (22.45%)

**分析**:
- DCC-GARCH和Basic GARCH方差降低效果相当，均超过84%
- ECM-DCC-GARCH方差降低仅为22%，远低于其他两个模型
- **原因**: ECM提取了误差修正项后，残差的相关性几乎消失(0.0123)，DCC部分无法发挥作用

### 2. 收益率表现

**排序**: Basic GARCH (0.04%) > DCC-GARCH (-0.26%) > ECM-DCC-GARCH (-2.26%)

**分析**:
- **Basic GARCH是唯一实现正收益的模型**
- DCC-GARCH小幅亏损(-0.26%)
- ECM-DCC-GARCH跟随市场(-2.26%接近未套保-2.58%)

### 3. 套保比例特征

| 模型 | 套保比例均值 | 稳定性 | 特征 |

| Basic GARCH | 0.6311 | 稳定 | 基于120天滚动相关系数 |

| ECM-DCC-GARCH | 0.0655 | 极低 | 协整主导，DCC失效 |

### 4. 相关性估计

**Basic GARCH**:
- 方法: 滚动窗口相关系数（120天）
- 均值: 0.9046
- 优点: 简单、稳健、响应平滑

- 问题: ECM提取了大部分相关性，残差几乎不相关

## 理论分析：为什么ECM-DCC-GARCH表现不佳？

### 原因1: ECM残差相关性消失

```
原始数据相关系数: ~0.90
↓
ECM提取协整关系后
↓
残差相关系数: 0.0123 (几乎为0)
```

**结论**: ECM部分已经提取了现货-期货的大部分相关性，残差几乎不相关，导致DCC部分无法发挥作用。

### 原因2: 套保比例公式失效

ECM-DCC-GARCH的套保比例：
```
h_t = h_ecm + 动态调整
```

当动态相关性为0时，动态调整部分失效，套保比例退化为ECM的静态比例。

**本数据**: 动态相关性0.0123 → 套保比例接近0.6424 × 调整系数 → 实际仅0.0655

### 原因3: 模型复杂度 vs 实际效果

ECM-DCC-GARCH虽然理论上更完善（考虑了协整+动态相关），但在本数据上：
- ECM部分: 提取了大部分相关信息（R²=0.93）
- DCC部分: 残差相关性极低，无法发挥作用
- **结果**: 复杂模型反而不如简单模型
